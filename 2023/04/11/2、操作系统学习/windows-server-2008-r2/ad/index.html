Active Directory域

Categories： Tags：

需要准备若干台机子，并配置好ip,确保在同一个网段


部署ad主服务器

添加服务器角色，选择ad域服务


点击下一步


点击安装


等待安装ing


点击关闭完成安装


打开服务器管理器，找到找到ad域服务，点击运行ad域服务安装向导


点击下一步


点击下一步


新建域


命名林根域，填写一个域名


等待安装


选择林功能级别（一般选择系统版本即可


安装dns服务器


选择继续


设置位置


设置还原模式密码


点击下一步


点击完成


重启计算机


打开ad域管理器，查看是否正在运行，ad部署完成


加入ad域

打开系统属性，点击更改


修改计算机名，隶属于更改为域，填写域名


输入主域的用户密码


加入成功


点击是


关闭窗口


重启计算机


打开服务器管理器，即可看到
