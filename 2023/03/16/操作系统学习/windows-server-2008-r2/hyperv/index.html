安装

1、给虚拟机开启虚拟化

在虚拟机的cpu选项寻找vt-x/amd-v，并启动

2、打开服务器管理器


点击角色 -> 添加角色 -> 服务器角色 -> Hyper-v -> 下一步


3、选择网卡创建虚拟网络（打钩即可）


4、重启服务器


5、关闭即可
